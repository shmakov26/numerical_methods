# Постановка для 4 задачи

Рассматривается обыкновенное дифференциальное уравнение второго порядка  
$$\ddot{x} + p(t) \dot{x} + q(t)x = f(t)$$
Необходимо найти решение $x = x(t)$ данного уравнения, удовлетворяющее краевым условиям  
$$\alpha_1 x(a) + \beta_1 \dot{x}(a) = \gamma_1, \quad \alpha_2 x(b) + \beta_2 \dot{x}(b) = \gamma_2.$$
Будем считать, что данная краевая задача имеет единственное решение.
Рассмотрим набор линейно независимых функций $\varphi_1(t), \varphi_2(t), \dots, \varphi_N(t)$, удовлетворяющих краевым условиям. Будем искать приближение решения в виде  
$$x_N(t) = C_1 \varphi_1(t) + C_2 \varphi_2(t) + \dots + C_N \varphi_N(t).$$
Введем в рассмотрение скалярное произведение $\langle f, g \rangle = \int_a^b f(t)g(t)\,dt$ и линейный оператор $L(x) = \ddot{x} + p(t) \dot{x} + q(t)x$. Для определения коэффициентов $C_1, C_2, \dots, C_N$ воспользуемся методом Галеркина.


**Необходимо написать программу**, которая находит приближенное решение краевой задачи методом Галеркина. Для вычисления производных в операторе $L$ необходимо использовать приближенные формулы
$$\dot{x}(t) \approx \frac{x(t+h) - x(t-h)}{2h},$$
$$\ddot{x}(t) \approx \frac{x(t-h) - 2x(t) + x(t+h)}{h^2},$$
где $h$ — малая положительная константа. Для вычисления интегралов используйте квадратурную формулу, погрешность которой не хуже квадратичной. При вводе пользователь должен вводить количество функций $N$. На выходе программа должна выводить график функции $x_N(t)$.

$$x(0) = 0, \quad \dot{x}(1) = 0, \quad \varphi_i(t) = \sin(i\pi t)$$
